# Syllabus
## Basic Java Programming
### Java Fundamentals
* Basic Syntax and Structure
* Variables and data types, references vs primitives...
* Operators
* Control Flow
* Strings
* Basic Looping
* Static Methods

### Object Oriented Programming
* Introduction to OOP - Importance of abstraction and encapsulation
* Classes and Objects
* Members, access modifiers, constructors...
* Methods
* Static keyword
* Basic Polymorphism, with an emphasis on Compositions and Interfaces
* Basic OOP patterns - Singleton, Dependency Injection...
