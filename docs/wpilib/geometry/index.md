# Introduction to Geometry In Robotics

Love it or hate it, Geometry is extremely important in robotics.

At it's core, robotics is the practice of turning code into physical motion, and space is the medium it moves through.

Software can make decisions, but it cannot move a mechanical part without knowing exactly where.

In this section, we will go over basic geometric representations, and how we can use and manipulate them in code.
