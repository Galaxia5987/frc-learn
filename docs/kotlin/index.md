# Introduction to Kotlin

Kotlin is an open source statically typed programming language that targets the JVM([Java Virtual Machine](https://en.wikipedia.org/wiki/Java_virtual_machine), basiclly the program that runs the Java code after compiling). It's developed by [JetBrains](https://jetbrains.com), the same company that provides Intellij IDEA, our IDE.

Because Kotlin targets the JVM, the same way Java does, everything written in Java is compatible with Kotlin, and vice-versa.

## Kotlin Advantages

I like to think of Kotlin as an abstraction to Java(that's not entirely true in practice, but it's a nice way to think of it).
It's just a better, concise, maintainable, way to write Java code. Instead of writing hundreds of lines of error-prone Java code, Kotlin may be able to simplify and shorten that.

Kotlin does provide numerous new features that Java doesn't have, but we need to be careful there. A lot of the times, we should ask ourselves the following question - Why doesn't Java have this feature?
Sometimes, we will find that a cool Kotlin feature can make performance much worse, without being obvious about it. That doesn't mean that we shouldn't use Kotlin only features, we just need to be careful.
