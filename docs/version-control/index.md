# Introduction to Version Control

Have you ever found yourself doing a big school project, having many tries at it. At the end, you'll most likely found yourself with folder filled with files named `project_final.docx`, `project_final_v2.docx`, and `project_final_v2_PLEASE_USE_THIS_ONE_FINAL_FINAL_V2.docx`. Maybe you even tried to tackle this project with a friend, and I'm sure it became a version management hell.
If so, you already understand the need for version control. In our code, where we write hundreds of lines of code, and working multiple people as a team - keeping track of changes is important.

In the next documents, we will learn the most popular tool for version control, called Git.
