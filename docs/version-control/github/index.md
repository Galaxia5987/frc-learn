# What is GitHub?

GitHub is a cloud-based platform where we can store, share, and work together with others to write and store code.

Storing our code on GitHub lets us:

* Store the code in a safe place, so it never gets lost
* Let multiple people synchronize and work on the code at the same time
* Track and manage changes to our code over time.
* Let other team members or mentors review our code
